    显示方式

       元素都有自己默认的显示方式
         块级元素：从上到下，可以设置宽高，如 h1 p div 等
         行内元素：从左到右，不能设置宽高，如span，a
         行内块： 从左只有，可以设置宽高，如input img等
       除了默认显示效果之外，可以使用 display 属性来修改元素框的显示方式，即改变生成的框类型
         display: none 不显示该元素，释放其占用位置
         display:block 将元素显示方式设置为块，如可以将行内元素 a 转换为块元素
         display:inline 将元素显示方式设置为行内 如可以将块元素 p 转换为行内元素
         display:inline-block 将元素显示方式设置为行内块 如可以将行内块元素 span 转换为行内块元素
