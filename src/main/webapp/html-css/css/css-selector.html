背景色：
   background-color 给元素设置背景接受任何合法颜色值
背景图片:
   background-imager:url('图片地址');
     默认值为none,表示背景上没有任何元素 可以通过url(设置图片)
   background-repeat 可以控制背景图片的效果 可取值
     1.repeat 在垂直方向和水平方向重复      2.repeat-x 仅在水平方向重复      3.repeat-y 仅在垂直方向重复      4.no-repeat 仅显示一次
   background-position 用于改变图片在元素中的位置
      background-position: 50% 50% 取X轴和Y轴 百分比移动
      background-position: left center right top bottom 在页面或者包含元素的左边显示、中间显示、右边显示、顶部显示、底部显示
    background-attachment: 改变图片是否跟随文档移动
      background-position: fixed 背景图像固定       默认情况下，背景会随文档滚动 scroll
